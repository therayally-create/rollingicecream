# Rolling Ice Cream

Public site for Rolling Ice Cream — small-batch, hand-rolled ice cream.

Live preview: https://therayally-create.github.io/rollingicecream/

## Local development

Open `index.html` in a browser. No build step.

## Files

- `index.html` — the entire site
- `fonts/` — Paula Natalie (wordmark), Transcity DEMO (display)
- `frames/` — 243 morph frames extracted from the source video
